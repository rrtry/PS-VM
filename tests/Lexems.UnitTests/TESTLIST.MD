# Список тестов лексера

## 1. Идентификаторы и ключевые слова

### 1.1 Простые идентификаторы
`x y z123 _underscore`

### 1.2 Все ключевые слова
`fn let return if else while for break continue true false str bool int float unit`

### 1.3 Неверный токен
`.x`

## 2. Литералы
### 2.1 Числовые литералы (разные системы счисления) и отдельный случай на float (3.)
`255 0xFF 0xff 0b11111111 3.14 0.5 3,14 3. 4`

### 2.2 Смешанный код с литералами
`let x = 3.14; let s = "text";`

### 2.3 Строковые литералы (пустые и непустые)
`"" "0" "Hello, world!"`

### 2.4 Строковый литерал с текстом
`let hello = "Hello world";`

### 2.5 Строка, не закрытая кавычкой
`let hello = "Hello world;`

### 2.6 Escape-последовательности в строке
`"\n\t\"\\"`

## 3. Escape-последовательности и комментарии (`GetEscapeSequencesAndCommentsData`)
### 3.1 Различные пробельные символы
`x \t\ny`

### 3.2 Неизвестная escape-последовательность
`\\q`

### 3.3 Однострочный комментарий
`let x = 0; // This is a comment`

### 3.4 Многострочные комментарии
`/* comments */ a / /* should be */ b * c /* ignored */`

## 4. Выражения и операторы
### 4.1 Оператор остатка от деления
`x % y`

### 4.2 Оператор возведения в степень
`x ** y`

### 4.3 Сложное арифметическое выражение
`x + y / (10 - z * 2)`

### 4.4 Логические операторы (ИЛИ)
`a < b || a > c || b == c`

### 4.5 Логические операторы (И) и (НЕТ)
`a <= b && b >= c && a != c && !b`

### 4.6 Побитовые (И) и (ИЛИ) отсутствуют в языке
`& |`

## 5. Конструкции языка

### 5.1 Объявление функции
`fn add(a: int, b: int): int { return a + b; }`

### 5.2 Условный оператор if-else
`if (x > 100) { return 100; } else { return x; }`

### 5.3 Цикл for
`for (let i = 1; i <= 5; i = i + 1) { printi(i); }`

### 5.4 Цикл while с break
`while (n < 100) { if (n > 10) { break; } n = n * 2; }`
