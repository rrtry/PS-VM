# Итерация #2:

[Lexer] Лексический анализ:
1. Написать лексер, который распознает ключевые слова (fn, let), числа, строки в кавычках, операторы и скобки

[AST] Синтаксический анализ (парсер) и AST:
1. Реализовать парсер для построения AST для программ с функцией main, арифметическими выражениями, строковыми операциями и вызовом print()

[CodeGen] Генерация байт-кода для литералов и вывода:
1. Написать генератор кода, который обходит AST и создает последовательность инструкций для VM, загружающую константы (числа, строки) и вызывающую PRINT

[VM] Реализация базовых инструкций VM:
1. Реализовать в VM выполнение инструкций для работы с числами и строками: загрузка констант , вывод, базовые операции над строками.

[User] Вывод всех типов литералов:
1. Пример: Программа print(42); print(3.14); print("Hello, World!"); выводит "42", "3.14" и "Hello, World!" каждое на новой строке.

[User] Декомпозиция и длина строк:
1. Пример (длина): printi(strlen("abcdef"));              // Выводит "6"
2. Пример (декомпозиция): printi(substr("abcdef", 1, 4)); // Выводит "bcd"

[User] Арифметические выражения с целыми числами:
1. Пример: Программа printi((2 + 3) * 4); выводит "20".

[User] Арифметические выражения с float:
1. printf(5.0 / 2.0, 2); выводит "2.5".
